# Employee Management System

A full-stack Employee Management System built with Java Spring Boot, React, and MySQL. 
This application allows you to manage employees, including adding, updating, deleting, and viewing employee information.

## Features

- Employee CRUD operations
- RESTful API built with Spring Boot
- Frontend built with React.js
- MySQL database integration
- Clean project structure with separate frontend and backend
- Secure configuration using .gitignore

## Navigation

- Home Page – Overview or welcome screen
- Employees List – View all employees with edit/delete options
- Add Employee – Form to add a new employee
- Edit Employee – Update employee details

## Tech Stack

Backend: Java Spring Boot  
Frontend: React.js  
Database: MySQL  
Build Tool: Maven  
Version Control: Git & GitHub

## Project Structure

backend/      # Spring Boot backend  
employeefrontend/  # React frontend  
.gitignore        # Root ignored files

## Setup Instructions

### Backend
1. Navigate to backend folder: `cd backend`
2. Run: `./mvnw spring-boot:run`
3. Ensure MySQL is running and update application.properties

### Frontend
1. Navigate to frontend folder: `cd employeefrontend`
2. Install dependencies: `npm install`
3. Run frontend: `npm start`
